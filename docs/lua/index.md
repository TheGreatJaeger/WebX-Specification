# Lua
The programming language of WebX.

## Versions
The lua api is separated in versions.\
These can be set like:
```html
<script api="<version>" src="...">
```
If no version provided will default to legacy

## Support
| API Version | Legacy | v2  |
| :---------- | :----: | :-: |
| Napture     | Yes    | No  |
| Bussinga    | Yes    | No  |
| YAB         | Yes    | Yes |
